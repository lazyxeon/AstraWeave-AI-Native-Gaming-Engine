## Core Architecture

```
Perception → Reasoning → Planning → Action
    ↓           ↓           ↓          ↓
WorldSnapshot  AI Model   PlanIntent  Tool Validation
```

AstraWeave's AI-native architecture treats every NPC as an intelligent agent running through a validated perception → planning → action loop. The engine supports classical AI (behavior trees, GOAP, utility curves), LLM-powered planning (Qwen3-8B via Ollama), ensemble voting, and hybrid arbiter configurations with a 4-tier fallback system.

## Subsystems

### 🎮 [Entity Component System](ecs.html)
Archetype-based ECS with deterministic 7-stage system pipeline, typed event channels, and ordered entity iteration. 25.8 ns world creation, <1 ns/entity tick. Miri-validated (330 tests).

### 🧠 [AI Orchestration](ai.html)
7-mode AI: Classical, BehaviorTree, Utility, LLM, Hybrid, Ensemble, and Arbiter. 12,700+ agents at 60 FPS with 6.48M anti-cheat validations/sec.

### 🎨 [Rendering Pipeline](rendering.html)
wgpu 25.0.2 renderer with PBR materials (Cook-Torrance BRDF), clustered forward lighting, cascaded shadow maps, post-processing (bloom, tonemapping), GPU skinning, GPU particle system, deferred path, LOD generation, and vertex compression.

### ⚡ [Physics Simulation](physics.html)
Rapier3D 0.22 wrapper with character controllers, spatial hashing (99.96% pair reduction), ragdolls, vehicles, cloth, gravity zones, destruction, fluids (2,404 SPH tests), and a 3-stage parallel pipeline.

### 🗺️ [Navigation](navigation.html)
Triangle-based navmesh baking with A* search, walkability filtering, slope analysis, dynamic region invalidation, and partial rebaking. 216 tests.

### 🔊 [Audio](audio.html)
Spatial audio via rodio 0.17 with 4-bus mixer (Master/Music/SFX/Voice), 3D panning, crossfading, dialogue runtime, and TTS adapter. 239 tests.

### 🖥️ [UI Framework](ui.html)
egui 0.32 in-game UI: menus, HUD (health bars, damage numbers, minimap, quest tracker), dialogue trees, accessibility features, gamepad navigation. 331 tests.

### 🎬 [Gameplay Systems](gameplay.html)
Combat physics (raycast sweep with parry/i-frames), crafting, quests, dialogue trees, items, biomes, and Veilweaver fate-weaving mechanic (94.26% coverage). 471 tests.

### 🌍 [Scene Management](scene.html)
Hierarchical scene graph with transform hierarchy, world partitioning, async cell streaming via `WorldPartitionManager`, and GPU resource management. 210 tests.

### 🏔️ [Terrain](rendering.html)
Voxel/polygon hybrid with dual contouring, SIMD noise, biome blending, hydraulic/thermal/wind erosion, LOD management, and AI-orchestrated dynamic terrain. 2,536 tests.

### 📐 [SIMD Math](math.html)
glam-based vector/matrix/quaternion ops with SSE2 acceleration: 1.7–2.5× speedup over scalar. Miri-validated (109 tests, 0 UB).

### 🔌 [C SDK](sdk.html)
Stable C ABI via cbindgen for embedding in non-Rust applications. Full world lifecycle, JSON snapshot/intent API. Miri-validated (17 tests).

## Getting Started

### Prerequisites

- **Rust 1.89.0+** (nightly recommended for Miri validation)
- **Windows/Linux/macOS** (wgpu supports Vulkan, DX12, Metal)
- **Optional**: Ollama with Qwen3-8B model for LLM features

### Quick Build

```bash
# Clone
git clone https://github.com/lazyxeon/AstraWeave-AI-Native-Gaming-Engine.git
cd AstraWeave-AI-Native-Gaming-Engine

# Build (first build takes 15-45 min for wgpu dependencies)
cargo build

# Run tests
cargo test-all

# Run the AI demo (7 modes: Classical, BehaviorTree, Utility, LLM, Hybrid, Ensemble, Arbiter)
cargo run -p hello_companion --release
```

### Cargo Aliases

```bash
cargo check-all    # Workspace check with exclusions
cargo build-core   # Core components only
cargo test-all     # Tests on working crates
cargo clippy-all   # Full linting with -D warnings
```
