## Core Architecture

```
Perception → Reasoning → Planning → Action
    ↓           ↓           ↓          ↓
WorldSnapshot  AI Model   PlanIntent  Tool Validation
```

AstraWeave's AI-native architecture treats every NPC as an intelligent agent running through a validated perception → planning → action loop. The engine supports classical AI (behavior trees, GOAP), LLM-powered planning (Hermes 2 Pro via Ollama), and hybrid configurations.

## Subsystems

### 🎮 [Entity Component System](ecs.html)
Archetype-based ECS with deterministic system stages, event handling, and ordered entity iteration. 25.8 ns world creation, <1 ns/entity tick.

### 🧠 [AI Orchestration](ai.html)
Multi-mode AI: classical (behavior trees, GOAP), LLM (Hermes 2 Pro), and hybrid arbiter. 12,700+ agents validated at 60 FPS.

### 🎨 [Rendering Pipeline](rendering.html)
wgpu 25 renderer with PBR materials, clustered forward lighting, CSM shadows, post-processing (bloom, TAA, DoF), GPU particles, deferred path, and mesh optimization.

### ⚡ [Physics Simulation](physics.html)
Rapier3D 0.22 wrapper with character controllers, spatial hashing, ragdolls, vehicles, cloth, gravity zones, destruction, and a 3-stage parallel pipeline.

### 🗺️ [Navigation](navigation.html)
Triangle-based navmesh baking with A* search, walkability filtering, slope analysis, and dynamic region invalidation.

### 🔊 [Audio](audio.html)
Spatial audio via rodio with 4-bus mixer, 3D panning, dialogue runtime, and TTS adapter.

### 🖥️ [UI Framework](ui.html)
egui-based in-game UI: menus, HUD (health bars, minimap, quest tracker, dialogue), accessibility (colorblind modes), gamepad support.

### 🎬 [Gameplay Systems](gameplay.html)
Combat (raycast sweep with parry/i-frames), crafting, quests, dialogue trees, items, biomes, and Veilweaver-specific mechanics.

### 🌍 [Scene Management](scene.html)
Hierarchical scene graph with world partitioning, async cell streaming, and GPU resource management.

### 📐 [SIMD Math](math.html)
Platform-optimized vector/matrix/quaternion ops with SSE2/AVX2 acceleration: 1.7–2.5× speedup over scalar.

### 🔌 [C SDK](sdk.html)
Stable C ABI for embedding in non-Rust applications via cbindgen. Full world lifecycle, JSON snapshot/intent API.

## Getting Started

### Prerequisites

- **Rust 1.89.0+** (nightly recommended for Miri)
- **Windows/Linux/macOS** (wgpu supports Vulkan, DX12, Metal)
- **Optional**: Ollama with Hermes 2 Pro model for LLM features

### Quick Build

```bash
# Clone
git clone https://github.com/YOUR_ORG/AstraWeave-AI-Native-Gaming-Engine.git
cd AstraWeave-AI-Native-Gaming-Engine

# Build (first build takes 15-45 min for wgpu dependencies)
cargo build

# Run tests
cargo test-all

# Run the AI demo
cargo run -p hello_companion --release
```

### Cargo Aliases

```bash
cargo check-all    # Workspace check with exclusions
cargo build-core   # Core components only
cargo test-all     # Tests on working crates
cargo clippy-all   # Full linting
```
