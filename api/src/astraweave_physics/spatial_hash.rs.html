2# Spatial Hash Grid for Broad-Phase Collision Detection
3
4Implements a grid-based spatial partitioning system to reduce collision detection
5from O(n²) to O(n log n) by only testing objects in nearby grid cells.
6
7## Performance
8
9**Baseline (Naive O(n²))**:
10- 1000 entities: ~500,000 collision pairs to test
11- Frame time: 548.5 µs (17.71% of frame)
12
13**With Spatial Hashing**:
14- 1000 entities: ~5,000-10,000 collision pairs (99% reduction)
15- Expected frame time: 250-330 µs (8-10% of frame)
16- **Improvement**: -40-55% collision detection time
17
18## Usage
19
20```rust
21use astraweave_physics::SpatialHash;
22
23// Create grid with 10-unit cells (adjust based on object sizes)
24let mut grid = SpatialHash::new(10.0);
25
26// Insert objects
27for (id, aabb) in objects {
28    grid.insert(id, aabb);
29}
30
31// Query potential collisions for an object
32let candidates = grid.query(aabb);
33
34// Only test narrow-phase collision against candidates (not all objects)
35for candidate_id in candidates {
36    if detailed_collision_test(aabb, candidate_aabb) {
37        // Handle collision
38    }
39}
40
41// Clear grid before next frame
42grid.clear();
43```
44
45## Cell Size Selection
46
47**Rule of Thumb**: Cell size should be ~2× the average object size
48
49- Too small: Objects span multiple cells, redundant queries
50- Too large: Too many objects per cell, approaches O(n²)
51- Optimal: Each object touches 1-4 cells
52
53**Example**:
54- Character radius: 1 unit → Cell size: 2-4 units
55- Small objects (0.5 radius) → Cell size: 1-2 units
56- Large objects (5 radius) → Cell size: 10-20 units
57*/

61/// Axis-Aligned Bounding Box for collision detection
62#[derive(Debug, Clone, Copy)]
63pub struct AABB {
64    pub min: Vec3,
65    pub max: Vec3,
66}
67
68impl AABB {
69    /// Create AABB from center and half-extents
70    pub fn from_center_extents(center: Vec3, half_extents: Vec3) -> Self {
71        Self {
72            min: center - half_extents,
73            max: center + half_extents,
74        }
75    }
76
77    /// Create AABB from center and radius (sphere approximation)
78    pub fn from_sphere(center: Vec3, radius: f32) -> Self {
79        let half_extents = Vec3::splat(radius);
80        Self::from_center_extents(center, half_extents)
81    }
82
83    /// Check if two AABBs intersect
84    pub fn intersects(&self, other: &AABB) -> bool {
85        self.min.x <= other.max.x
86            && self.max.x >= other.min.x
87            && self.min.y <= other.max.y
88            && self.max.y >= other.min.y
89            && self.min.z <= other.max.z
90            && self.max.z >= other.min.z
91    }
92
93    /// Get center point of AABB
94    pub fn center(&self) -> Vec3 {
95        (self.min + self.max) * 0.5
96    }
97
98    /// Get half-extents of AABB
99    pub fn half_extents(&self) -> Vec3 {
100        (self.max - self.min) * 0.5
101    }
102}

104/// Grid cell coordinates (3D integer grid)
105type GridCell = (i32, i32, i32);
106
107/// Spatial hash grid for broad-phase collision detection
108///
109/// Uses a uniform grid to partition 3D space. Objects are inserted into grid cells
110/// based on their AABB. Queries return only objects in nearby cells, dramatically
111/// reducing the number of collision pairs to test.
112///
113/// **Phase B Optimization (Action 27)**: Uses FxHashMap for faster hashing.
114/// Week 9 testing confirmed FxHashMap provides better performance than SipHash
115/// for spatial grids (3.77ms vs 5.61ms with Tracy, 3.82ms without Tracy).
116#[derive(Debug)]
117pub struct SpatialHash<T> {
118    /// Grid cell size (world units)
119    cell_size: f32,
120    
121    /// Inverse cell size (for faster division-free grid coordinate calculation)
122    inv_cell_size: f32,
123    
124    /// Grid storage: (x, y, z) → [object IDs]
125    /// Uses FxHashMap for faster non-cryptographic hashing
126    grid: rustc_hash::FxHashMap<GridCell, Vec<T>>,
127    
128    /// Total objects currently in grid
129    object_count: usize,
130}
131
132impl<T: Copy + Eq + Ord> SpatialHash<T> {
133    /// Create new spatial hash with specified cell size
134    ///
135    /// # Arguments
136    /// * `cell_size` - Grid cell size in world units (recommended: 2× average object size)
137    ///
138    /// # Example
139    /// ```
140    /// let grid = SpatialHash::<u32>::new(10.0); // 10-unit cells
141    /// ```
142    pub fn new(cell_size: f32) -> Self {
143        assert!(cell_size > 0.0, "Cell size must be positive");
144        
145        Self {
146            cell_size,
147            inv_cell_size: 1.0 / cell_size,
148            grid: rustc_hash::FxHashMap::default(),
149            object_count: 0,
150        }
151    }
152
153    /// Convert world position to grid cell coordinates
154    #[inline]
155    fn world_to_cell(&self, pos: Vec3) -> GridCell {
156        (
157            (pos.x * self.inv_cell_size).floor() as i32,
158            (pos.y * self.inv_cell_size).floor() as i32,
159            (pos.z * self.inv_cell_size).floor() as i32,
160        )
161    }
162
163    /// Get all grid cells that an AABB overlaps
164    fn get_overlapping_cells(&self, aabb: &AABB) -> Vec<GridCell> {
165        let min_cell = self.world_to_cell(aabb.min);
166        let max_cell = self.world_to_cell(aabb.max);
167
168        let mut cells = Vec::new();
169        
170        // Iterate over all cells that the AABB spans
171        for x in min_cell.0..=max_cell.0 {
172            for y in min_cell.1..=max_cell.1 {
173                for z in min_cell.2..=max_cell.2 {
174                    cells.push((x, y, z));
175                }
176            }
177        }
178
179        cells
180    }
181
182    /// Insert object into grid based on its AABB
183    ///
184    /// Objects that span multiple cells are inserted into all overlapping cells.
185    ///
186    /// # Arguments
187    /// * `id` - Object identifier
188    /// * `aabb` - Object's axis-aligned bounding box
189    ///
190    /// # Example
191    /// ```
192    /// let aabb = AABB::from_sphere(Vec3::new(5.0, 0.0, 5.0), 1.0);
193    /// grid.insert(entity_id, aabb);
194    /// ```
195    pub fn insert(&mut self, id: T, aabb: AABB) {
196        let cells = self.get_overlapping_cells(&aabb);
197        
198        for cell in cells {
199            self.grid
200                .entry(cell)
201                .or_default()
202                .push(id);
203        }
204        
205        self.object_count += 1;
206    }
207
208    /// Query potential collision candidates for an AABB
209    ///
210    /// Returns all objects in grid cells that overlap the query AABB.
211    /// Duplicates are possible if objects span multiple cells.
212    ///
213    /// # Arguments
214    /// * `aabb` - Query bounding box
215    ///
216    /// # Returns
217    /// Vec of object IDs that *might* collide (candidates for narrow-phase testing)
218    ///
219    /// # Example
220    /// ```
221    /// let query_aabb = AABB::from_sphere(pos, radius);
222    /// let candidates = grid.query(query_aabb);
223    /// 
224    /// for &candidate_id in &candidates {
225    ///     // Perform detailed collision test with candidate
226    /// }
227    /// ```
228    pub fn query(&self, aabb: AABB) -> Vec<T> {
229        let cells = self.get_overlapping_cells(&aabb);
230        let mut results = Vec::new();
231
232        for cell in cells {
233            if let Some(objects) = self.grid.get(&cell) {
234                results.extend_from_slice(objects);
235            }
236        }
237
238        results
239    }
240
241    /// Query unique collision candidates (removes duplicates)
242    ///
243    /// Same as `query()` but deduplicates results. Slightly slower due to sorting,
244    /// but useful when duplicate checks are expensive.
245    pub fn query_unique(&self, aabb: AABB) -> Vec<T> {
246        let mut results = self.query(aabb);
247        results.sort_unstable();
248        results.dedup();
249        results
250    }
251
252    /// Clear all objects from grid
253    ///
254    /// Call this before rebuilding grid each frame (dynamic objects).
255    pub fn clear(&mut self) {
256        self.grid.clear();
257        self.object_count = 0;
258    }
259
260    /// Get number of objects in grid
261    pub fn object_count(&self) -> usize {
262        self.object_count
263    }
264
265    /// Get number of occupied grid cells
266    pub fn cell_count(&self) -> usize {
267        self.grid.len()
268    }
269
270    /// Get grid cell size
271    pub fn cell_size(&self) -> f32 {
272        self.cell_size
273    }
274
275    /// Get average objects per cell (density metric)
276    pub fn average_cell_density(&self) -> f32 {
277        if self.grid.is_empty() {
278            0.0
279        } else {
280            self.object_count as f32 / self.grid.len() as f32
281        }
282    }
283
284    /// Debug: Get grid statistics
285    pub fn stats(&self) -> SpatialHashStats {
286        let mut max_objects_per_cell = 0;
287        let mut total_objects_in_cells = 0;
288
289        for objects in self.grid.values() {
290            max_objects_per_cell = max_objects_per_cell.max(objects.len());
291            total_objects_in_cells += objects.len();
292        }
293
294        SpatialHashStats {
295            object_count: self.object_count,
296            cell_count: self.grid.len(),
297            max_objects_per_cell,
298            average_objects_per_cell: if self.grid.is_empty() {
299                0.0
300            } else {
301                total_objects_in_cells as f32 / self.grid.len() as f32
302            },
303            cell_size: self.cell_size,
304        }
305    }
306}
